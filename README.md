# PRM Portafolio Academico

Proyecto en PHP + MySQL con arquitectura hexagonal.

## Setup
1) Copia `app/Infrastructure/Config/env.example` a `.env`
2) Ejecuta `composer install`
3) Ejecuta el SQL `app/Infrastructure/Persistence/Migrations/001_init.sql` en phpMyAdmin
4) Abre: http://localhost/prm_portafolio/public/
