# PRM Portafolio Académico

Este repositorio corresponde al proyecto actual en **Django**.

## Nota de mantenimiento
La carpeta `app/` pertenecía a una versión anterior (PHP) y fue retirada para evitar confusión.

## Ejecución local (Django)
1. Crear y activar entorno virtual.
2. Instalar dependencias del proyecto.
3. Ejecutar migraciones.
4. Levantar servidor:

```bash
python manage.py runserver
```
